- [x] 壁纸功能
- [ ] 极简的名言模式
- [ ] 壁纸下支持书签

- [x] 搜索引擎切换、搜索建议、搜索框
- [ ] 待办事项
- [ ] 类 typora 的编辑器
- [ ] Maybe some funny tools 😂
